**firmngin.dev** adalah platform **AIoT** (Artificial Intelligence \+ Internet of Things) untuk bisnis perangkat keras. Satu tempat untuk menghubungkan firmware ke cloud, memantau perangkat secara real-time, menjalankan otomasi, dan mengelola layanan berbayar — dari prototipe hingga produksi.

## Apa itu firmngin.dev?

Firmngin menggabungkan **manajemen perangkat**, **firmware & OTA**, **otomasi berbasis aturan**, dan **monetisasi** dalam satu alur kerja. Anda mendaftar workspace, mendaftarkan board (misalnya ESP32/ESP8266), mengintegrasikan library resmi, lalu mengoperasikan perangkat lewat dashboard — tanpa merakit banyak layanan terpisah.

Platform ini cocok untuk pembuat produk IoT, tim hardware, dan bisnis yang ingin menjual akses perangkat atau layanan metered langsung dari firmware.

## Apa yang bisa Anda lakukan?

- **Menghubungkan perangkat** — generate kredensial MQTT, publish telemetri/state, dan lihat perubahan langsung di dashboard atau WebSocket.
- **Mengontrol & mengotomasi** — atur entity, jalankan rule/automation, dan hubungkan event perangkat ke aksi cloud atau perangkat lain.
- **Mendeploy firmware** — siapkan artifact OTA dan pantau rollout ke fleet perangkat.
- **Menjual layanan perangkat** — buat merchant (sandbox & live), katalog item, tautkan perangkat ke merchant, dan buka menu publik untuk pengguna akhir.
- **Membangun integrasi** — gunakan REST API, webhook, MQTT topics, dan dokumentasi API untuk sistem Anda sendiri.
