Use monetization callbacks when the device needs to react to paid orders, PIN verification, or active service sessions.

[image:paid IoT session flow from checkout to device active service]

<Note>
  Firmware should react to the typed events it receives. It should not expose internal broker topics, raw device secrets, or backend-only identifiers to end users.
</Note>
