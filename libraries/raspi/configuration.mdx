## Quick setup

```python
from firmngin import ClientConfig

config = ClientConfig.from_file("keys.json")
```

Use this config with `AsyncClient` (async) or `Client` (sync).

## KeysConfig

```python
from firmngin import KeysConfig

keys = KeysConfig.from_file("keys.json")
```

### Fields

| Field | Type | Description |
| --- | --- | --- |
| `device_id` | `str` | Device identifier |
| `device_key` | `str` | Authentication key |
| `decryptor` | `str` | Hex-encoded AES key for E2EE |
| `validation_mode` | `str` | `"ca"`, `"pin"`, or `"both"` |
| `ca_cert` | `str \| None` | CA certificate PEM |
| `service_ca_cert` | `str` | Service CA PEM |
| `client_cert` | `str \| None` | Client cert for mTLS |
| `private_key` | `str \| None` | Private key for mTLS |
| `fingerprint_sha256` | `str \| None` | SHA-256 fingerprint |

The `decryptor_key` property holds the parsed key bytes.

## ClientConfig

| Field | Type | Description |
| --- | --- | --- |
| `keys` | `KeysConfig` | Device credentials (required) |

## Debug logging

```python
async with AsyncClient(config) as client:
    client.set_debug(True)
    await client.connect()
```
