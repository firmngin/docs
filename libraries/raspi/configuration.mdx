## Quick setup

```python
from firmngin import ClientConfig

config = ClientConfig.from_file("keys.json")
```

Gunakan config ini dengan `AsyncClient` (async) atau `Client` (sync).

## KeysConfig

```python
from firmngin import KeysConfig

keys = KeysConfig.from_file("keys.json")
```

### Fields

| Field | Type | Description |
| --- | --- | --- |
| `device_id` | `str` | Device identifier |
| `device_key` | `str` | Authentication key |
| `decryptor` | `str` | Hex-encoded AES key untuk E2EE |
| `validation_mode` | `str` | `"ca"`, `"pin"`, atau `"both"` |
| `ca_cert` | `str \| None` | CA certificate PEM |
| `service_ca_cert` | `str` | Service CA PEM |
| `client_cert` | `str \| None` | Client cert untuk mTLS |
| `private_key` | `str \| None` | Private key untuk mTLS |
| `fingerprint_sha256` | `str \| None` | SHA-256 fingerprint |

Property `decryptor_key` menyimpan byte key yang sudah di-parse.

### Load from environment

```python
keys = KeysConfig.from_env(prefix="FIRMNGIN_")
```

## ClientConfig

| Field | Default | Description |
| --- | --- | --- |
| `keys` | (required) | `KeysConfig` |
| `queue_path` | `None` | Offline queue path |
| `queue_max_size` | `500` | Batas jumlah pesan queue |
| `queue_max_bytes` | `5242880` | Batas ukuran queue (bytes) |
| `connect_timeout_seconds` | `10.0` | MQTT timeout |
| `keepalive_seconds` | `60` | MQTT keepalive |
| `reconnect_max_attempts` | `None` | Batas reconnect (`None` = unlimited) |
| `mtls` | `True` | Mutual TLS |

Broker MQTT dikelola platform (`asia-jkt1.firmngin.dev:58884`).

## TLS modes

| Mode | Use case |
| --- | --- |
| `"ca"` | Production — validasi CA |
| `"pin"` | Fingerprint pinning |
| `"both"` | Default |

## Debug logging

```python
async with AsyncClient(config) as client:
    client.set_debug(True)
    await client.connect()
```
