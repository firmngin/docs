## Async client

```python
import asyncio
from firmngin import AsyncClient, ClientConfig, Entity, Event, Payment

async def on_payment(payment: Payment) -> None:
    if payment.is_success:
        print("paid:", payment.order_id)

async def main():
    config = ClientConfig.from_file("keys.json")
    async with AsyncClient(config) as client:
        client.on(Event.PAYMENT, on_payment)
        await client.connect()
        await client.push_entity(Entity("relay"), "on")
        await client.run()

asyncio.run(main())
```

## Sync client

```python
from firmngin import Client, ClientConfig, Entity

with Client(ClientConfig.from_file("keys.json")) as client:
    client.connect()
    client.push_entity(Entity("relay"), "off")
```

Handler async pada sync client: `client.async_client.on(...)`

## Entity command

```python
from firmngin import Entity, EntityCommand

@client.on_entity(Entity(1))
async def handle_relay(command: EntityCommand) -> None:
    print(command.value)
```
