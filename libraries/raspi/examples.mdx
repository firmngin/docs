## Basic async client

```python
import asyncio
from firmngin import Client, ClientConfig, Entity, Event, Payment

async def on_payment(payment: Payment) -> None:
    if payment.is_success:
        print("paid:", payment.order_id)

async def main():
    config = ClientConfig.from_file("keys.json")
    async with Client(config) as client:
        client.on(Event.PAYMENT, on_payment)
        await client.connect()
        await client.push_entity(Entity("relay"), "on")
        await client.run()

asyncio.run(main())
```

## Entity command handler

```python
from firmngin import Entity, EntityCommand

relay = Entity(1)

@client.on_entity(relay)
async def handle_relay(command: EntityCommand) -> None:
    if command.value == "on":
        print("relay on")
    elif command.value == "off":
        print("relay off")
```

## Sync client

Untuk skrip blocking tanpa `asyncio`:

```python
from firmngin import ClientConfig, Entity, SyncClient

with SyncClient(ClientConfig.from_file("keys.json")) as sync:
    sync.connect()
    sync.push_entity(Entity("relay"), "off")
```

Akses async client di dalam wrapper: `sync.client.on(...)`

## Batch and location

```python
batch = client.push_batch_entities()
batch.add(Entity(1), "on")
batch.add(Entity("2"), 27.5)
await batch.send()

await client.push_location_values(lat=-6.2, lon=106.816666, accuracy=8.5)
```
