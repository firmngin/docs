Daftarkan handler pada `AsyncClient` dengan `client.on(Event.X, handler)`.

## Event types

| Event | Payload |
| --- | --- |
| `Event.PAYMENT` | `Payment` |
| `Event.PAYMENT_PENDING` | `Payment` |
| `Event.PAYMENT_SUCCESS` | `Payment` |
| `Event.INIT` | `Init` |
| `Event.MERCHANT_STATUS` | `str` |
| `Event.VERIFICATION` | `Verification` |
| `Event.PIN` | `Verification` |
| `Event.DEVICE_STATUS` | `DeviceStatus` |
| `Event.ENTITY_COMMAND` | `EntityCommand` |
| `Event.ACTIVE_SESSION` | `ActiveSession` |
| `Event.ERROR` | `Exception` |

## Register handlers

```python
from firmngin import AsyncClient, ClientConfig, Event, Payment

config = ClientConfig.from_file("keys.json")

async with AsyncClient(config) as client:
    @client.on(Event.PAYMENT)
    async def handle_payment(payment: Payment) -> None:
        if payment.is_success:
            print("paid:", payment.order_id)
```

Hapus handler:

```python
client.off(Event.PAYMENT, handle_payment)
client.clear_handlers()
```

## Active session

```python
@client.on(Event.ACTIVE_SESSION)
async def handle_session(session: ActiveSession) -> None:
    if session.can_run:
        await session.end_session()
```

## Error handling

```python
@client.on(Event.ERROR)
async def handle_error(error: Exception) -> None:
    print("error:", error)
```
