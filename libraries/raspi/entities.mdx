## Entity patterns

```python
from firmngin import Entity

relay = Entity(1)              # GPIO pin / numeric key
temperature = Entity("temperature")  # string key dari dashboard
```

## Push state

```python
await client.push_entity(relay, "on")
await client.push_entity(temperature, 27.5)
```

## Batch entities

```python
batch = client.push_batch_entities()
batch.add(relay, "on")
batch.add(temperature, 27.5)
await batch.send()
```

## Entity commands

Per entity:

```python
from firmngin import EntityCommand

@client.on_entity(relay)
async def handle_relay(command: EntityCommand) -> None:
    print("relay:", command.value)
```

Global:

```python
from firmngin import Event

@client.on(Event.ENTITY_COMMAND)
async def handle_command(command: EntityCommand) -> None:
    print(command.key, command.value)
```
