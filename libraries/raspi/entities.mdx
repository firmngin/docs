---
title: "Entities and State"
description: "Send device state and receive entity commands from Firmngin"
---

Entities are the user-facing values and controls exposed by a device. Use them for sensor readings, relay state, status values, counters, and dashboard commands.

## Define entities

Use an `Entity` object when you want a typed reference to an entity key.

```python
from firmngin import Entity

relay = Entity("relay")
temperature = Entity("temperature")
status = Entity(1)
```

## Push state

Use `push_entity()` for a single value update. This is useful for sensors, relay feedback, or any state that changes over time.

```python
await client.push_entity(temperature, 25.5)
await client.push_entity(relay, "on")
await client.push_entity(status, 1)
```

Boolean values are automatically serialized: `True` becomes `"1"`, `False` becomes `"0"`.

## Batch updates

For multiple values at once, use the batch builder:

```python
batch = client.push_batch_entities()
batch.add(relay, "on")
batch.add(temperature, 27.5)
batch.add("humidity", 80)
await batch.send()
```

You can also use `update_entities()` directly:

```python
await client.update_entities({
    "relay": "on",
    "temperature": 27.5,
    "humidity": 80,
})
```

## Receive commands

Use `on_entity()` to receive commands sent from Firmngin to the device. The command includes an entity key and value, so the application can route it to the correct handler.

```python
from firmngin import EntityCommand

relay = Entity("relay")

@client.on_entity(relay)
async def handle_relay(command: EntityCommand) -> None:
    print("relay command:", command.value)
```

You can also use the `on_entity()` method directly without the decorator:

```python
async def handle_relay(command: EntityCommand) -> None:
    print("relay command:", command.value)

client.on_entity(relay, handle_relay)
```

For multiple entity commands, register one global handler and switch by key:

```python
@client.on(Event.ENTITY_COMMAND)
async def handle_command(command: EntityCommand) -> None:
    if command.key == "relay":
        print("relay:", command.value)
    elif command.key == "temperature":
        print("temperature:", command.value)
```

## Read local entity values

Read the last known value of an entity from local state:

```python
value = client.entity(temperature)
print(value.to_string())
print(value.to_float())
print(value.to_int())
print(value.is_on())
```

## Request init

Call `request_init()` when the application needs the latest device configuration after connection or reconnect.

```python
await client.connect()
await client.request_init()
await client.run()
```

## Location updates

Use the location builder for GPS tracking:

```python
await client.push_location_values(
    lat=-6.208763,
    lon=106.845599,
    accuracy=5.0,
    alt=10.0,
    speed=1.2,
)
```

Or use the builder step by step:

```python
location = client.push_location()
location.lat(-6.208763)
location.lon(106.845599)
location.accuracy(5.0)
await location.send()
```

## Image upload

Upload an image and link it to an entity:

```python
camera = Entity("camera")
await client.upload_image(camera, "snapshot.jpg")
```
