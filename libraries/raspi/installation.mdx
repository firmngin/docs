## Install the package

From Git:

```bash
pip install git+https://github.com/firmngin/firmngin-sdk-python.git
```

From PyPI after release:

```bash
pip install firmngin
```

## Device credentials

Device identity and cryptographic material are loaded from `keys.json`. Download the keys file from the device dashboard and place it next to your script.

```python
from firmngin import ClientConfig, KeysConfig

config = ClientConfig(keys=KeysConfig.from_file("keys.json"))
```

Do not commit real device keys to version control.

## Development requirements

The following tools are used for development:

- `ruff` — linting and formatting
- `mypy --strict` — type checking
- `pytest` — unit and integration tests
- `pip-audit` — dependency vulnerability scanning
- `bandit` — security analysis
