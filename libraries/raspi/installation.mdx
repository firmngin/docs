## Install the package

From PyPI:

```bash
pip install firmngin
```

From Git:

```bash
pip install git+https://github.com/firmngin/firmngin-sdk-python.git
```

### Optional: image upload

```bash
pip install firmngin[http]
```

## Dependencies

| Package | Required | Purpose |
| --- | --- | --- |
| `aiomqtt` | Yes | MQTT client |
| `cryptography` | Yes | TLS + AES-GCM E2EE |
| `httpx` | No (`[http]` extra) | Multipart image upload |

Sisanya memakai Python stdlib — tanpa Pydantic atau framework config eksternal.

## Device credentials

Unduh `keys.json` dari device dashboard Firmngin. Letakkan di direktori kerja script Anda.

```python
from firmngin import ClientConfig

config = ClientConfig.from_file("keys.json")
```

Jangan commit `keys.json` yang berisi kredensial nyata.

## Development requirements

Tools yang dipakai saat pengembangan SDK:

- `ruff` — lint dan format
- `mypy --strict` — type checking
- `pytest` — unit dan integration tests
- `pip-audit` — vulnerability scan
- `bandit` — security analysis
