Gunakan Firmngin Python SDK untuk Raspberry Pi, Linux-based devices, integration tests, dan server-side tooling.

## Features

- Event-driven async API dengan `Client`
- TLS, mTLS, dan certificate fingerprint validation
- E2EE (AES-GCM) aktif secara default
- Typed payloads: payment, init, verification, device status, entity commands
- Entity publishing, batch updates, location, active sessions, offline queue
- Image upload opsional via `firmngin[http]`

## Requirements

- Python 3.9+
- File `keys.json` dari dashboard Firmngin

## Dependencies

Instalasi inti hanya menarik **dua** library pihak ketiga:

| Package | Fungsi |
| --- | --- |
| `aiomqtt` | Transport MQTT |
| `cryptography` | TLS + enkripsi E2EE |

Upload gambar membutuhkan extra: `pip install firmngin[http]`

## Minimal lifecycle

```python
import asyncio
from firmngin import Client, ClientConfig, Entity

relay = Entity(1)
temperature = Entity("temperature")

async def main():
    config = ClientConfig.from_file("keys.json")
    async with Client(config) as client:
        await client.connect()
        await client.push_entity(relay, "on")
        await client.push_entity(temperature, 27.5)
        await client.run()

asyncio.run(main())
```
