# Raspberry Pi (Python SDK)

Gunakan Firmngin Python SDK untuk Raspberry Pi, Linux-based devices, integration tests, dan server-side tooling.

## Installation

```bash
pip install git+https://github.com/firmngin/firmngin-sdk-python.git
```

## Entity patterns

Entity key wrapper works with both patterns:

```python
from firmngin import Entity

# By GPIO pin — matches the "gpio_pin" field in dashboard
relay = Entity(1)

# By entity key — matches the "key" field in dashboard
temperature = Entity("temperature")
```

## Push state

```python
import asyncio
from firmngin import ClientConfig, Entity, FirmnginClient, KeysConfig

relay = Entity(1)
temperature = Entity("temperature")

async def main():
    config = ClientConfig(keys=KeysConfig.from_file("keys.json"))
    async with FirmnginClient(config) as client:
        await client.connect()
        await client.push_entity(relay, "on")
        await client.push_entity(temperature, 27.5)
        await client.run()

asyncio.run(main())
```

## Batch entities

```python
batch = client.push_batch_entities()
batch.add(relay, "on")
batch.add(temperature, 27.5)
await batch.send()
```

## Entity commands

Register per-entity command handlers:

```python
from firmngin import EntityCommand

@client.on_entity(relay)
async def handle_relay(command: EntityCommand) -> None:
    print("relay:", command.value)
```

Or use global event listener:

```python
from firmngin import Event

client.on(Event.ENTITY_COMMAND, lambda cmd: print(cmd.key, cmd.value))
```

## Events

Supported event types:

| Event | Trigger |
| --- | --- |
| `Event.PAYMENT` | Payment pending or success |
| `Event.INIT` | Initial device config |
| `Event.PIN` | PIN verification |
| `Event.DEVICE_STATUS` | Device lifecycle changes |
| `Event.ENTITY_COMMAND` | Entity command from dashboard |
| `Event.ACTIVE_SESSION` | Active paid session |

## Image upload

```python
camera = Entity("camera_image")
await client.upload_image(camera, "snapshot.jpg")
```

## Sync adapter

For blocking scripts:

```python
from firmngin import SyncFirmnginClient

client = SyncFirmnginClient(config)
client.connect()
client.push_entity(relay, "off")
```
