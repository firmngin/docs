Gunakan Firmngin Python SDK untuk Raspberry Pi, Linux-based devices, integration tests, dan server-side tooling.

## Features

- Event-driven async device client API
- TLS, mTLS, and certificate fingerprint validation
- E2EE enabled by default
- Typed payload models for payment, init, verification, device status, and entity commands
- Entity state publishing, batch updates, location updates, active-session helpers, and image upload

## Requirements

- Python 3.9 or newer
- A platform-issued `keys.json` file for the device

## Minimal lifecycle

```python
import asyncio
from firmngin import ClientConfig, Entity, FirmnginClient, KeysConfig

relay = Entity(1)
temperature = Entity("temperature")

async def main():
    config = ClientConfig(keys=KeysConfig.from_file("keys.json"))
    async with FirmnginClient(config) as client:
        await client.connect()
        await client.push_entity(relay, "on")
        await client.push_entity(temperature, 27.5)
        await client.run()

asyncio.run(main())
```
