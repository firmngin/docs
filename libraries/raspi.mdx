Gunakan Firmngin Python SDK untuk Raspberry Pi, Linux-based devices, integration tests, dan server-side tooling.

## Features

- `AsyncClient` — event-driven async API
- `Client` — synchronous blocking wrapper
- TLS, mTLS, dan certificate fingerprint validation
- E2EE (AES-GCM) aktif secara default
- Typed payloads, entity publishing, batch, location, offline queue
- Image upload opsional via `firmngin[http]`

## Requirements

- Python 3.9+
- File `keys.json` dari dashboard Firmngin

## Dependencies

| Package | Fungsi |
| --- | --- |
| `aiomqtt` | Transport MQTT |
| `cryptography` | TLS + enkripsi E2EE |

Upload gambar: `pip install firmngin[http]`

## Minimal lifecycle (async)

```python
import asyncio
from firmngin import AsyncClient, ClientConfig, Entity

relay = Entity(1)
temperature = Entity("temperature")

async def main():
    config = ClientConfig.from_file("keys.json")
    async with AsyncClient(config) as client:
        await client.connect()
        await client.push_entity(relay, "on")
        await client.push_entity(temperature, 27.5)
        await client.run()

asyncio.run(main())
```
