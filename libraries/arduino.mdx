Use the Arduino library for firmware that connects ESP-class boards to firmngin.

## Use this library for

- New ESP32 or ESP8266 devices.
- Devices that publish structured entity values.
- Devices that need command handling from the dashboard.
- Firmware that must report availability and reconnect reliably.

## Core responsibilities

The firmware integration should authenticate with device credentials, connect to MQTT, publish entity values, subscribe to command topics, report lifecycle state, and reconnect after network or broker interruption.
