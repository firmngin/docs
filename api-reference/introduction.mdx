Rules:

- Use `error`, not `code` or `error_code`.
- Use `UPPER_SNAKE_CASE` error codes.
- Return human-readable messages.
- Never expose raw internal errors to clients.

## Common endpoint groups

| Group | Example path | Purpose |
| --- | --- | --- |
| Auth | `/auth/login` | Login, OTP, OAuth, profile session. |
| Workspaces | `/workspaces` | Workspace selection and team scope. |
| Devices | `/devices`, `/devices/{device_id}` | Device registry and detail. |
| Device entities | `/device-entities` | Entity values, logs, grouped entity data. |
| Dashboards | `/dashboards`, `/dashboards/device/{device_id}` | Workspace dashboards and device overview dashboards. |
| Firmware | `/firmware...` | Firmware artifacts, releases, rollout state. |
| Automations | `/automations/graphs` | Automation graph CRUD and execution data. |
| Orders | `/orders`, `/orders/{order_id}/stop-session` | Monetization order and session management. |
